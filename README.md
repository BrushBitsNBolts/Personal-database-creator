# Editing or deleting stuff:
Inorder to edit a text object you must right click the text object and click "edit text" which then will open a text area which allows for text editing.
You can only have 1 text area open at once. To close a text area simply press f2.
You may have noticed the delete button as well, it does exactly what it sounds like, it takes the current object and removes it completely.
# Saving and loading
To save or load you have to exit edit mode, inorder to exit edit mode simply click "enter edit mode" again.
